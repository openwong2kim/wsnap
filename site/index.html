단축키 한 번 → 영역 드래그 → 떠오른 썸네일을 그대로 끌어다 놓기.

3단계

눌렀다, 그렸다, 끌어다 놓는다.

저장 위치를 고를 일도, 폴더를 뒤질 일도 없습니다. 캡처에서 사용까지 끊김 없이 이어집니다.

STEP 01
Shift+F1

단축키를 누른다

기본 Shift+F1. 어떤 키 조합으로도 바꿀 수 있고, 캡처 모드가 즉시 켜집니다.

STEP 02

영역을 드래그한다

원하는 범위를 마우스로 끌어 선택합니다. 픽셀 단위 치수가 실시간으로 표시됩니다.

STEP 03

그대로 끌어다 놓는다

오른쪽 아래 썸네일을 채팅·업로드·탐색기로 드래그하면 진짜 파일이 떨어집니다. 클릭하면 경로 복사.

핵심 기능

썸네일이 곧 파일입니다.

캡처하면 오른쪽 아래에 떠오르는 작은 카드. 그것을 잡아끄는 순간, 그곳에 알맞은 것이 떨어집니다. 임시 저장도, 복사-붙여넣기 왕복도 없습니다.

  • 탐색기·채팅·업로드 칸으로 끌면 실제 파일이 떨어집니다이미지를 첨부받는 어떤 입력란에도 그대로
  • 터미널로 끌면 파일 경로가 입력됩니다커맨드라인 작업이 한층 빨라집니다
  • 그냥 클릭하면 경로가 클립보드에 복사됩니다
  • 오른쪽으로 휙 던지면 사라지고, 시간이 지나면 자동으로 정리됩니다
탐색기 / 폴더 → 파일 .png
채팅 / 업로드 칸 → 첨부됨
터미널 → C:\…\snap.png

모든 기능

캡처 그 이상.
하루의 작업 흐름을 위해.

읽고, 만들고, 다듬는 일까지 — 전부 한 도구 안에서, 전부 당신의 PC 안에서.

영역 캡처 + 썸네일 스택

드래그로 영역을 잡으면 오른쪽 아래에 카드가 쌓입니다. 최대 개수는 설정에서 조절. 호버하면 편집·OCR·삭제 버튼이 나타나고, 오른쪽 드래그로 던져 버립니다.

드래그앤드롭 출력

썸네일을 끌면 실제 파일이, 터미널에선 경로가 떨어집니다. 클릭하면 경로 복사. wsnap의 첫 번째이자 핵심 기능입니다.

PRIMARY

온디바이스 OCR

PaddleOCR PP-OCRv5 엔진으로 한국어·영어 혼합 텍스트를 정확하게 인식합니다. 캡처에서 바로 텍스트를 복사하거나, 전용 영역 OCR 모드로 화면 속 글자만 긁어냅니다. 모델이 내장돼 언어팩도 필요 없고, 네트워크 없이 무료로.

PaddleOCR · 한국어 + 영어 · 오프라인

미니 에디터

화살표·사각형·펜·텍스트·모자이크/블러·자르기. 키보드 우선 설계로 즉시 전환하고, 실행 취소를 지원합니다.

ARPTMC

GIF 녹화

화면의 한 영역을 그대로 녹화해 반복 재생되는 애니메이션 GIF로 저장합니다. 짧은 동작을 보여주기에 딱 맞습니다.

스크롤 캡처

자동으로 휠을 굴려 화면을 따라 내려가며 여러 장을 찍고, 하나의 길쭉한 이미지로 이어 붙입니다. 긴 페이지도 한 장에.

클립보드 감시

다른 도구로 복사한 이미지도 감지해 자동으로 썸네일을 띄웁니다. 어디서 온 캡처든 같은 흐름으로 다룰 수 있습니다.

자유로운 단축키 지정

저수준 키보드 후킹으로 원하는 어떤 조합이든 바인딩합니다. 시스템 캡처 단축키 Win+Shift+S 가로채기도 선택할 수 있습니다.

정확한 멀티모니터 캡처

PerMonitorV2 DPI 인식으로 서로 다른 배율·소수점 스케일링·여러 모니터 환경에서도 흐림 없이 픽셀 정확하게 잡아냅니다.

PerMonitorV2 DPI

상주 필수 기능

트레이 메뉴, 윈도우 시작 시 자동 실행, 단일 인스턴스 보장, 설정 패널, 그리고 충돌 로그 기록까지 — 늘 조용히 대기합니다.

로컬·온디바이스·추적 없음

모든 처리가 당신의 PC 안에서 일어납니다. 외부 전송도, 추적도 없습니다. 텔레메트리는 선택 사항이며, 켜더라도 로컬에만 남습니다.

PRIVACY-FIRST

선택형 Imgur 업로드

원하면 직접 발급한 키로 Imgur에 올려 링크를 받을 수 있습니다. 기본은 꺼져 있으며, 켜는 것도 끄는 것도 전적으로 당신의 선택입니다.
